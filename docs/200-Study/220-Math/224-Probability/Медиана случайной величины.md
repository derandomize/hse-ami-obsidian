# Определение
$\xi$ - величина
$m \in \R$ - ее медиана, если
$P(\xi \geq m) = \frac{1}{2}$ и $P(\xi \leq m) \geq \frac{1}{2}$
# Пример 1
$\xi = 1,2 ,3 ,4 ,5, 6$  c вероятностью $\frac{1}{6}$, от 3 до 4 - медианы
# Пример 2
$1000$ человек все кроме одного получают $1000$ долларов, начальник $1000000$.

$$E = 1999$$
$m = 1000$
