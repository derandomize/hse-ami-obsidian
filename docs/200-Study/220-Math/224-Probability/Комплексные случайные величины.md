# Определение
$\xi: \Omega \to \C$
комплекснозначный случайные величины, если $\text{Re } \xi$ $\text{Im } \xi$ - случайные величины
# Матожидание
$$E\xi := E\text{ Re } \xi + i E\text{ Im }\xi$$
## Свойства
1. $a \in \R$ $E(a\xi) = a E \xi$
   $E(i\xi) = i E \xi$
   $E(c\xi) = c E \xi$ 
2. $E\overline{\xi} = \overline{E\xi}$
3. $|E \xi| \leq E |\xi|$
